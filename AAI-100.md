# AAI-100: Concept Specification

## Purpose

AAI-100 defines the concept and guiding principles for the Android Autonomy Interface.

## Scope

AAI provides a standardized interface between Android-based autonomy software and embedded vehicle controllers.

## Design Principles

- Vehicle-agnostic autonomy
- Hardware discovery on connection
- Safety-limit negotiation
- Testable before hardware
- Plug-and-play vehicle profiles
